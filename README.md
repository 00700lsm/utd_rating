# ⚽UtdRating
**UtdRating**은 맨체스터 유나이티드의 경기기록과 평점을 기록하고 직관적으로 볼수 있는 프로그램입니다.

## 데이터베이스 구조
![PL Match](https://github.com/user-attachments/assets/529cafa1-9b89-4796-9091-a2ead42e3e32)

## 주요 기능
### 선수 등록
### 경기 등록
### 경기별 선수 평점 등록

## 기술 스택
-   **Kotlin**
-   **Spring Boot**
-   **MySQL**
-   **JPA**
